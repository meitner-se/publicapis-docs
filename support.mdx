## Frequently Asked Questions

<AccordionGroup>
  <Accordion title="Do I need to pay extra for API access?">
    No. If you're a Meitner client, API access is included at no additional cost. We believe you own your data and should have free access to it.
  </Accordion>
  
  <Accordion title="What's the difference between client and partner API access?">
    Clients get free API access to their own organization's data. Partners get dedicated API keys for building integrations that work with multiple organizations.
  </Accordion>
  
  <Accordion title="How long does it take to get API keys?">
    For existing clients, API keys are typically provided within 1-2 business days. For new partnerships, the process may take 1-2 weeks including review and agreement.
  </Accordion>
  
  <Accordion title="Are there any usage limits?">
    No artificial usage limits for clients accessing their own data. Partners may have reasonable limits during development and testing phases.
  </Accordion>
  
  <Accordion title="Can I use the API for commercial purposes?">
    Clients can use their API access for any purpose related to their organization. Partners must comply with our partnership agreement terms.
  </Accordion>
</AccordionGroup>
